  • SymfoSeries

    Web application

    • PHP
    • Symfony
    • phpMyAdmin
    • Bootstrap

    In the context of my studies, I built with a friend a website that shows a list of series coming from "imdb.com". Two main parts were developed, the first one is meant for a client, and the second one for an administrator. As a client, you can create an account, log in, then search for series and episodes and add them to your watch list, rate them etc... As an administrator, you can add a new show with the IMDB API and manage all the client's comments.

  • Dedisclasik

    Desktop application

    • C# .NET
    • Winforms
    • Microsoft SQL Server

    In the context of my studies, I built with a team of 6 developers a software able to manage a vinyl collection of a store. In 6 days, we worked in a SCRUM environment with 2 days sprints to deliver to our "client" (our teachers) the solution of his dreams. Developed using C# .NET Core, Winforms and Microsoft SQL Server Management Studio for the database, we managed to deliver a working application while learning a lot concerning team and project management.
